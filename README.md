# 🚖 Uber Ride Analytics Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze Uber ride operations, customer behavior, revenue trends, and business performance.

The dashboard was built using Power BI, DAX, Power Query, and Excel to transform raw ride data into actionable business insights.

---

## 🚀 Dashboard Features

- Executive Overview Dashboard
- Operations Analytics Dashboard
- Business Insights Dashboard
- Interactive Slicers
- Navigation Buttons
- KPI Cards
- Revenue Trend Analysis
- Peak Hour Heatmap
- Driver Rating Analysis
- Customer Rating Analysis
- Cancellation Analysis
- Map Visualization

---

## 🛠 Tools Used

- Power BI
- DAX
- Power Query
- Microsoft Excel

---

## 📊 Key Insights

- Pune generated the highest revenue.
- Sedan was the most popular vehicle.
- Peak ride demand occurred around 8 AM.
- Revenue remained stable throughout the year.
- Customer and driver ratings averaged around 4.0.
